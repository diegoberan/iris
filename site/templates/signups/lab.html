🚀 Unordered Future Ideas

A raw list of features, experiments, and wild integrations I want to build next into the Íris ecosystem:

  • Even Realities G2 smart glasses integration: Expose a head-up display capability. If the Even Realities team is reading this: please open an API! I'd love to stream agent alerts directly onto my HUD.
  • Wear OS Voice synthesis: Run a lightweight F5-TTS cloned voice model locally on target devices, generating audio responses directly on the watch.
  • Local device triggers: Allow local scripts to trigger actions based on hardware status (e.g. flashing keyboard Morse code when the orchestrator performs a failover).
  • Sarcastic Notification Listener: Build an Android capability that reads incoming notifications aloud in a playful, completely custom, sarcastic tone.
