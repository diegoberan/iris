AMD compute, demonstrably

Your GPU first. AMD cloud when you need it.

One OpenAI-compatible route lets the Brain use the closest capable hardware, then advance when a local body goes offline.

Local

RX 9060 XT

Gemma runs beside Dot's F5-TTS voice on the user's own Radeon — private, available, and first in line.

AMD Cloud

MI300 · vLLM

The current cloud runtime hosts Gemma on an AMD MI300 pod when local capacity is unavailable or scale matters.

Fireworks AI

A tested resilience route.

The Fireworks API is wired as the last-resort route. Stop a local body and the next turn advances without losing the Brain.

Validated across the stack

Íris was also tested on the hackathon's Radeon PRO W7900 pod, serving Gemma Q4 through llama.cpp. The project therefore proves the same routing idea across local Radeon, AMD cloud via vLLM, workstation-class AMD hardware, and the Fireworks AI API.
